getpass

The getpass module provides two functions:

  • getpass.getpass([prompt, [,stream]]): very useful when asking for password without echoing. The user is prompted using the string prompt, which defaults to 'Password: '. E.g., 
                password = getpass.getpass('>> type in ur pass: ')
  • getpass.getuser() returns the login user of system. Equals to Linux command whoami. 
                print(getpass.getuser())
