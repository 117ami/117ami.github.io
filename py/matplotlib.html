Pyplot Tutorial

Let's start with a a simple draw with Pyplot.

plot() is a versatile command, and will take an arbitrary number of arguments. For example, to plot x versus y, we can issue the command: 

    		plt.plot([1, 2, 3, 4], [1, 4, 9, 16])
    		# plt.plot([1, 2, 3, 4], [1, 4, 9, 16], 'ro')
Note that, there is an optional third argument which is the format string that indicates the color and line type of the plot. For example, 'ro' in the above code means red circles.
